Mega-question

Github Page

Within a form, can an overall question be programmatically associated with a series of form controls and their labels below? These test pages demonstrate three different methods.

  • Example 1: Using aria-describedby
  • Example 2: Using nested fieldsets
  • Example 3: Using multiple instances of aria-describedby

Results

A pass is defined as a radio button that is announced with:

  1. The form control label: "Yes"
  2. The parent legend: "Mould"
  3. The overall question: "Does your house have any of the following problems"
